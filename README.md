# Tanara

Tanana is a starter theme for bantenprov laravel application, with supported vuejs

## Install

### post release
```s
$ composer create-project bantenprov/tanara
```

### Development snapshot
```sh
## install npm
make sure you have latest nodejs or atleat nodejs 8
```
npm install
```

## run npm
make sure you have latest nodejs or atleat nodejs 8
```
npm run dev
```
or

```
npm run production
```
